## 📖 文章1: 平民流量手册（杨涛）

**基本信息**:
- 时间: 2026-02-01
- 作者: 杨涛（航海家）
- 字数: ~25,000字
- 点赞: 402赞 + 1830人点赞

### 核心框架

#### 一、占位思维（搜索 > 推荐）

**核心观点**:
> 主动搜索过来的客户品质最高，搜索流量价值远超推荐流量

**流量价值公式**:
```
1个搜索流量 = 10个表单流量 = 80个推荐流量（某些行业）
```

**关键词分类**:
1. **真实需求词**: 用户真正会搜的词
2. **伪需求词**: 看起来有人搜，但转化率低

**实操步骤**:
1. 理解客户画像
2. 研究关键词（长尾+精准）
3. 布局关键词策略
4. 占住入口

**案例**:
- "广州越秀小气泡" > "医美"
- "上海陆家嘴大闸蟹礼盒品牌哪家强" > "大闸蟹"

#### 二、四小时流量速成法

**适用场景**: 快速入局任何行业

**操作流程**:
1. **加同行 (3小时)**
   - 用用户视角，加30-50个同行
   - 观察：标题、封面、笔记、评论区、引流方式

2. **整理模板 (1小时)**
   - 发现5-8套重复模板
   - 选择最适合自己风格的模板

3. **收集资料**
   - 同行的话术、资料、问答
   - 整理成自己的行业百问百答

4. **关注高手**
   - 反复加到的肯定是高手
   - 深度模仿他的全套打法

**关键洞察**:
> "同行是最好的老师"
> "你加不到同行，客户就加不到你"

#### 三、场景-需求-产品思维

**选择即经营**

**思考框架**:
1. **场景**: 客户在什么场景下？
2. **需求**: 这个场景下有什么需求？
3. **产品**: 用什么产品/服务满足需求？

**案例**:
- 场景: 看病 → 需求: 陪诊、绿通、高端医疗 → 产品: 黄牛服务
- 场景: 遮丑 → 需求: 变美、变瘦、变体面 → 产品: 鲨鱼裤、束腰

**五高标准**:
- 高客单
- 高利润
- 高粘性
- 高复购
- 高转介绍

**核心理念**:
> "赚富人的零花钱，别赚穷人的生活费"

#### 四、流量结构思维

**健康流量结构**:
- 货架电商: 淘客 + 活动 + 综搜 + 硬广
- 知识付费: 混群 + 换量 + 裂变

**抗风险能力**:
- 单一流量来源 = 高风险
- 多元化流量 = 低风险

## 📖 文章2: 副业转创业决策（恒一）

**基本信息**:
- 时间: 2025-12-21
- 作者: 恒一（生财五年级）
- 字数: ~12,000字
- 厂年薪成绩: 大百万 → 副业日入过万 → 离职创业

### 核心决策框架

#### 一、创业前的三个准备

**1. 清空负债**
- 原因: 防止决策变形
- 行动: 提前规划还债时间表

**2. 项目可持续**
- 原因: 不能昙花一现
- 行动: 多项目并行，东方不亮西方亮
- 案例: 第二曲线 = 偶然发现 + 快速测试 + ROI验证

**3. 保持职场竞争力**
- 原因: 留好退路
- 行动: 离职前完成能力拼图
- 原则: 要可迁移的能力，不是平台的奖状

#### 二、两年系统性准备

**时间线**:
- 2021.07: 有创业想法，开始规划
- 2023.04: 完成所有准备目标
- 2023.04+: 继续待一年，利用平台资源

**具体行动**:
1. **主动接"边角料"项目**
   - 新渠道测试
   - 探索性项目
   - 牺牲短期绩效，换能力提升

2. **利用大厂身份**
   - 香港卡、招行金葵花
   - E类人才身份
   - 评标专家等荣誉

3. **建立人脉**
   - 潜在合伙人
   - 供应商
   - 行业资源

#### 三、创业后的教训

**1. 团队搭建过晚**
> "项目一旦跑通，第一要务不是自己离职，而是招募团队"

**2. 创业不改本性**
> "上班时让你别扭的事，创业后不会自动变好"

**关键认知**:
- 认清自己的短板
- 设计制度或找人补位

#### 四、发现项目的技巧

**1. 刷广告法**
- 关注: 消耗不高（<10w/月）+ 持续投放（>半年）
- 判断: 小需求 = 容易进入
- 行动: 点击广告，加微信，了解模式

**2. 观察生活法**
- 例子1: 买车时多问一句 → 发现优惠券项目 → 赚几台车的钱
- 例子2: 买房时发现痛点 → 轻松赚回房价跌幅

**3. 信息差法**
> "你遇到的问题，99%已被解决"
> "你所知的事，99%的人不知道"

**落地方法**:
- 留意"别人不知道的事"
- 留意"别人不知如何实现的事"
- 乘以14亿人口 = 足够市场

## 📖 文章3: AI改造业务实战（陈亮）

**基本信息**:
- 时间: 2025-08-11
- 作者: 陈亮（6年老圈友）
- 成绩: AI替代30人团队 + 50国内容运营

### AI改造业务四步法

#### 第一步：MVP验证

**核心**: 先小规模测试

**操作**:
1. AI接管部分环节
2. 双跑机制（AI + 人工）
3. 评估准确率和召回率

#### 第二步：逐步放权

**核心**: AI从辅助到核心

**操作**:
1. 人工审核AI结果
2. AI处理简单任务
3. 逐步增加AI权重
4. 最终AI主导

#### 第三步：工程化搭建

**核心**: 把流程变成机器

**操作**:
1. 流程跑通
2. 模块化
3. 工厂化
4. 稳定运转

#### 第四步：规模化复制

**核心**: 从1到N

**操作**:
1. 一个场景 → 多个场景
2. 一个国家 → 50个国家
3. 一个流程 → 整个业务

### 核心洞察

> "不是让AI适应业务，而是改造业务以适应AI"
