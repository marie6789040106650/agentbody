# 📊 信息来源标注规范

> **创建时间**: 2026-02-12
> **版本**: v1.0
> **适用范围**: 所有研究、调研、执行类任务

---

## 🎯 规范目的

确保所有产出的内容都有明确的数据来源，提高可追溯性和可信度。

---

## 📋 标注要求

### 必须包含的字段

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| **信息来源** | ✅ | 数据来源类型 | 实时抓取 / 知识库 / 混合 |
| **来源文件** | ✅ | 具体文件路径 | `scys/docs/04-方向研究/xxx.md` |
| **数据时效性** | ✅ | 数据新鲜度 | 实时 / 历史 / 过期 |
| **抓取时间** | ⏱ | 时间戳（仅实时抓取） | `2026-02-12 00:15` |
| **可信度评估** | ⚠️ | 数据质量评估 | 高 / 中 / 低 |

---

## 📝 文档模板

### 必须在文档头部添加

```markdown
> **文档信息**
> - 创建时间: YYYY-MM-DD HH:MM
> - 数据来源: [实时抓取/知识库/混合]
> - 主要来源: 具体文件路径或URL
> - 数据时效性: [实时/历史/过期]
> - [可选] 抓取时间: YYYY-MM-DD HH:MM
> - [可选] 可信度评估: [高/中/低]
```

---

## 📂 数据来源分类

### 1. 实时抓取 ✅

**标识**: `实时抓取`
**说明**: 通过工具实时抓取的最新数据
**可信度**: 高

**必须标注**:
- 抓取时间
- 抓取工具
- 抓取范围

**示例**:
```
数据来源: 实时抓取
主要来源: xiaohongshu API（关键词"AI电子书智能体"，近7天）
抓取时间: 2026-02-12 00:15
抓取工具: xiaohongshu-tools v1.0
可信度评估: 高
```

### 2. 知识库 ✅

**标识**: `知识库`
**说明**: 基于本地知识库的历史研究数据
**可信度**: 中（需标注数据生成时间）

**必须标注**:
- 知识库文件路径
- 原始研究时间

**示例**:
```
数据来源: 知识库
主要来源: scys/docs/04-方向研究/scys-three-directions-research.md
原始研究时间: 2026-02-10
可信度评估: 中（历史数据，建议实时验证）
```

### 3. 混合 ✅

**标识**: `混合`
**说明**: 实时抓取 + 知识库补充
**可信度**: 中高（需标注各部分来源）

**必须标注**:
- 各部分来源及比例
- 融合方式

**示例**:
```
数据来源: 混合
├── 70%: 实时抓取（xiaohongshu API，2026-02-12）
└── 30%: 知识库补充（scys-three-directions-research.md）
融合方式: 实时数据为主，知识库补充品类说明
可信度评估: 中高
```

---

## 🚫 禁止项

1. **禁止**无来源标注的结论性内容
2. **禁止**使用过期数据而不标注
3. **禁止**混合不同来源的数据而不说明
4. **禁止**使用无法追溯的工具输出

---

## ✅ 示例对比

### ❌ 错误示例

```markdown
> **调研时间**: 2026-02-11
> **调研范围**: 小红书平台
```

**问题**: 
- 未标注数据来源
- 未标注数据时效性
- 读者无法追溯数据可信度

### ✅ 正确示例

```markdown
> **调研时间**: 2026-02-11
> **调研范围**: 小红书平台虚拟产品赛道
> 
> **📊 数据来源说明**:
> - 数据来源: 知识库
> - 主要来源: scys/docs/04-方向研究/scys-three-directions-research.md
> - 原始研究时间: 2026-02-10（生财有术风向标）
> - 数据时效性: 历史数据（建议实时验证）
> - 可信度评估: 中
```

## 🔧 工具调用记录

### 必须记录的工具调用

| 工具 | 用途 | 必须记录 |
|------|------|---------|
| xiaohongshu | 竞品调研 | ✅ 调用时间、关键词、结果数 |
| web_search | 网络搜索 | ✅ 搜索时间、查询词、结果数 |
| web_fetch | 网页抓取 | ✅ 抓取时间、URL、内容摘要 |
| browser | 浏览器操作 | ✅ 操作时间、操作内容 |

```markdown

| 时间 | 工具 | 用途 | 结果 | 状态 |
|------|------|------|------|------|
| HH:MM | xiaohongshu | 竞品关键词搜索 | 20条笔记 | ✅成功 |
| HH:MM | web_search | 趋势分析 | 10条结果 | ✅成功 |
```
