```
高热度 + 高变现 = AI电子书智能体、副业攻略、合同文书

中热度 + 高变现 = AI提示词、简历模板
稳定型 = 思维导图、手账模板
```

| **低门槛** | Notion模板、节日文案、小红书模板、思维导图、手账 | 会整理+会排版+持续输出 |

## 三、标题/封面/内容模板库

### 3.1 爆款标题万能公式

#### 公式1：痛点+解决方案+成果
```
"[身份]必看｜[痛点]？用这个方法[解决方案]，效果[成果]"

```

#### 公式2：对比+数字+利益
```
"[时间/方式] vs [时间/方式]｜同样[任务]，[成果]差了[倍数]"
示例："手写3个月 vs AI 3天｜同样写书，产出差了10倍"
```

#### 公式3：身份+利益+情绪
```
"[身份]必备｜[利益承诺]🔥[情绪词]"
示例："运营人必备｜春节文案模板全齐了🔥爱了"
```

#### 公式4：数字+类型+免费
```
"[数字]个[类型]｜[特点]｜[行动词]"
示例："100个AI提示词｜免费领｜用完效率飞升"
```

#### 公式5：提问+回答+引导
```
"[问题]？用这个[产品/方法]，从0到1[结果]"
示例："电子书怎么写？用这个AI智能体，从0到1全搞定"
```

### 3.2 封面设计黄金法则

| 法则 | 说明 | 适用场景 |
|-----|------|---------|

| **信息层次** | 标题>副标题>装饰元素 | 所有品类 |
| **色彩统一** | 形成个人品牌色系 | Notion模板、手账 |

| **节日元素** | 融入节日符号 | 节日文案 |
| **工具展示** | 展示工具界面/效果 | AI工具、Notion |
| **成品展示** | 展示最终效果图 | 模板类、合集类 |

## 四、3个可复制变现模式

### 💰 模式1：模板销售型

#### 4.1.1 模式概述

```
核心逻辑: 整理/设计模板 → 笔记种草 → 店铺成交
适用品类: Notion模板、小红书模板、节日文案、手账模板
变现周期: 短（用户看到就会买）
```

#### 4.1.2 执行步骤

| 步骤 | 动作 | 时间投入 |
|-----|------|---------|
| **Step 1** | 选择细分领域，整理模板库 | 3-7天 |

### A.1 品类选择决策树

```
先问自己：
```
