## 1. 笔记标题生成器

### 模板A：悬念引导型

```markdown
## 模板A：悬念引导型

**适用场景**: 知识干货、技巧分享类内容

**使用说明**:
- 开头用"为什么"、"千万别"、"竟然"等词汇制造悬念
- 中间用数字强化信息量
- 结尾用"真相了"、"绝了"、"建议收藏"等收尾

**变量说明**:
- {主题}: 内容核心主题
- {痛点}: 用户关心的问题
- {数字}: 具体数字/数据
- {行动词}: 建议/方法/技巧

**模板结构**:
{悬念词} {主题} {数字} {行动词}，{痛点} {情绪词}！

**使用示例**:
输入:
 用- 主题:AI做副业
- 痛点: 没技能没人脉
- 数字: 3个方法
- 悬念词: 千万别
- 行动词: 亲测有效

输出:
- 千万别用AI做副业！除非你懂这3个方法，没技能没人脉也能月入过万（亲测有效）
- 为什么用AI做副业的人都在闷声发大财？3个方法让你从小白变大咖
```

## 2. 笔记正文生成器

### 类型A：知识干货型

```markdown
## 类型A：知识干货型

**适用场景**: 教程教学、方法论、技巧分享

**使用说明**:
- 开头: 痛点引入 + 解决方案预告
- 主体: 分步骤/分点讲解，每个点含"操作方法+效果"
- 结尾: 行动号召 + 互动引导

**变量**:
- {痛点}: 用户当前困境
- {主题}: 核心知识点
- {步骤数}: 步骤数量
- {核心技巧}: 关键方法
- {避坑点}: 常见误区

**模板结构**:

【开场】
{痛点}？今天分享{主题}，{步骤数}个技巧帮你{目标}。

【正文】
1. {技巧1}
   - 操作方法: 
   - 预期效果: 

2. {技巧2}
   - 操作方法: 
   - 预期效果: 

3. {技巧3}
   - 操作方法: 
   - 预期效果: 

【避坑指南】
⚠️ {避坑点1}
⚠️ {避坑点2}

【结尾】
以上就是{主题}的{步骤数}个技巧，建议{行动词}。
如果你觉得有用，记得{互动引导}！

---

**使用示例**:
输入:
- 痛点: 写笔记没人看
- 主题: 小红书爆款标题
- 步骤数: 3
- 核心技巧: 情绪词+数字+痛点
- 避坑点: 标题太正式、关键词堆砌

输出:
【开场】
笔记写了几十篇没人看？今天分享小红书爆款标题的3个技巧，帮你7天涨粉1000+。

【正文】
1. 情绪词开头
   - 操作方法: 在标题开头加入"千万别"、"太绝了"、"后悔没早知道"等情绪词
   - 预期效果: 点击率提升50%+

2. 数字强化
   - 操作方法: 用"3个"、"7天"、"100%"等具体数字
   - 预期效果: 增加可信度和信息量感知

3. 痛点+方案
   - 操作方法: 描述用户痛点 + 给出解决方案
   - 预期效果: 精准触达目标人群

【避坑指南】
⚠️ 标题太正式，像发论文
⚠️ 关键词堆砌，像打广告

【结尾】
以上就是小红书爆款标题的3个技巧，建议收藏起来用！
如果想要更多模板，评论区扣"666"领取！
```

## 使用指南

### 快速上手流程

```
1️⃣ 选择模板类型
   ├── 标题 → A/B/C 模板
   ├── 正文 → A/B/C 类型
   ├── 产品 → 产品描述模板
   └── 封面 → 封面文案模板

2️⃣ 填充变量
   └── 根据实际内容填入变量

3️⃣ AI生成初稿
   └── 使用 MiniMax-M2.1 或 Claude 生成

4️⃣ 人工优化
   └── 检查真实性、调整语气、添加个人特色
```

### 最佳实践

- **真实性**: 案例/数据需真实，夸大易翻车
- **个人特色**: 加入个人经历/语气，更易建立信任
- **互动设计**: 结尾留钩子（评论区扣1领资料）
- **数据验证**: 发布后跟踪数据，持续优化

### 常见问题

**Q: 生成的内容太AI味怎么办？**
A: 加入个人真实案例，用口语化表达，删减过渡句

**Q: 标题不够吸引人？**
A: 尝试不同风格模板，加入更强烈的情绪词

**Q: 内容太长/太短？**
A: 根据平台特性调整，小红书正文100-300字为佳
