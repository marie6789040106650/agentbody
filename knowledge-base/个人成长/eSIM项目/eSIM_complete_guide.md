## 📱 eSIM 安装方法

### 方法一: Trip.com App 内安装 (推荐 - iOS)
1. 确保 Trip.com App 更新至最新版本
2. iPhone 系统升级至 iOS 17.4 或更高版本
3. 打开 Trip.com 应用程式
4. 前往"账户" → "全部订单"
5. 点击"安装"按钮

### 方法二: 扫描 QR Code
1. 打开 Trip.com App 订单中的二维码附件
2. 提前打印或保存到另一台电子设备
3. 打开手机设置
4. iOS: "流动网络/行动服务" → "加入流动网络计划"
5. Android: "SIM卡管理" → "添加SIM卡管理"
6. 在手机联网状态下扫描二维码

### 方法三: 手动输入
如果无法扫描，订单确认 PDF 中包含：
- **SM-DP+Address**
- **Activation Code**

### 安装后配置
1. **关闭主卡** 的数据漫游
2. **启用 eSIM** 网络开关
3. **开启"数据漫游"** 选项

- **APN:** cmhk
- **iOS 设置路径:**
  - 设置 → 流动网络/行动服务 → 流动数据网络
- **Android 设置路径:**
  - 设置 → 网络和互联网 → 流动网络 → 存取点名称(APN) → 新建/编辑APN

## 📊 使用规则

### 日数计算
- 每24小时为一日
- 例：3日套餐，10月1日10:00安装
- 结束时间：10月4日09:59

### 提前安装
- ✅ 可以提前安装
- ✅ 抵达目的地后启动eSIM即开始计费

### 设备绑定
- ⚠️ 一个QR Code只能在一台手机上使用
- ⚠️ 安装后与设备绑定，无法转移
