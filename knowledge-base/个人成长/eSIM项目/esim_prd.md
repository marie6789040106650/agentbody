# 📱 中国通 eSIM - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 产品简介
**产品名称:** 中国通 eSIM (China Connect eSIM)

**产品定位:** 为出境旅客提供便捷的中国大陆/港澳台 eSIM 购买和安装服务

**核心价值:**
- 免换卡 - 扫码即用
- 免登记 - 无需护照信息
- 即时交付 - 邮件秒发

### 1.2 MVP 战略定位

```
┌─────────────────────────────────────────────────────────────────┐
│                      产品演进路线                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  MVP 阶段                                    扩展阶段            │
│  ┌─────────────────────┐                    ┌─────────────────┐ │
│  │  中国大陆 eSIM      │ ─── 业务跑通 ───▶ │  多地区 eSIM    │ │
│  │  Trip.com 单一采购  │                    │  多供应商对接   │ │
│  │  邮件发送           │                    │  API 聚合       │ │
│  └─────────────────────┘                    └─────────────────┘ │
│                                                                 │
│  目标: 验证商业模式                          目标: 扩大市场覆盖  │
│  时间: Week 1-4                              时间: Week 5+      │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.3 MVP 范围

| 维度 | MVP 阶段 | 扩展阶段 |
|------|----------|----------|
| **地区** | 中国大陆/港澳台 | 全球 50+ 国家和地区 |
| **供应商** | Trip.com (单一) | Trip.com + 其他 3-5 家 |
| **套餐** | 4-6 个固定套餐 | 50+ 动态套餐 |
| **支付** | Creem.io | Stripe + 本地支付 |
| **语言** | 简体中文 | 中/英/日/韩/泰 |
| **客服** | 邮件支持 | 7x24 在线客服 |

### 1.4 业务模式 (MVP)
```
用户下单 → Creem.io 支付 → Trip.com 采购 → 邮件发送 eSIM
```

### 1.5 目标用户 (MVP)
- 前往中国大陆/港澳台的旅客
- 商务出行人士
- 旅游观光游客
- 短期停留用户 (1-365天)

## 7. 风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| Trip.com 价格波动 | 利润减少 | 多源比价，选择最优供应商 |

## 8. 扩展路线图

### 8.1 MVP → 扩展阶段划分

```
Phase 1: MVP (Week 1-4)
├── 目标: 验证商业模式，跑通核心流程
├── 地区: 中国大陆/港澳台
├── 套餐: 4-6 个固定套餐
└── 重点: 支付、采购、邮件发送自动化

Phase 2: 扩展准备 (Week 5-8)
├── 目标: 优化用户体验，准备多地区接入
├── 新增: 日本、韩国、泰国、新加坡
└── 重点: 供应商 API 对接

Phase 3: 多地区上线 (Week 9-12)
├── 目标: 覆盖 10+ 地区，50+ 套餐
├── 新增: 欧洲、美洲、东南亚
└── 重点: 多供应商聚合系统

Phase 4: 全球化 (Week 13+)
├── 目标: 全球覆盖，100+ 地区
└── 重点: 品牌升级，市场扩张
```
