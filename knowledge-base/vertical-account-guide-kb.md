## 🎯 核心方法论速查

### 定位公式
> 我是（身份），专注帮（目标人群）在（具体场景）解决（核心问题），并用（方法/路径）让他们（获得最终结果）。

### 内容定位 4 种思路
1. **热点解读定位法** - 已有行业经验
2. **专业分享定位法** - 有专业技能
3. **成长养成系定位法** - 边学边分享
4. **爆款赛道定位法** - 从爆款切入

### 赛道选择 5 维度
1. 市场竞争烈度
2. 需求聚焦度
3. 用户画像精准度
4. 解决方案壁垒
5. 变现路径清晰度

### 变现 5 模式
1. 流量直接变现（流量主）
2. 流量 + 别人产品（分销）
3. 流量 + 自己产品（课程/咨询）
4. 别人流量 + 自己产品（投放）
5. 别人流量 + 别人产品（渠道代理）

## 📊 数据分析指标

| 指标 | 定义 | 衡量标准 |
|------|------|----------|
| **点击率** | 粉丝看到文章后点击打开的比例 | <1%危险，>5%优秀 |
| **完读率** | 阅读用户从头到尾的比例 | <20%需优化，>40%优秀 |
| **关注率** | 阅读后关注公众号的比例 | >0.5%优秀 |
| **转发率** | 阅读后分享收藏的比例 | >3%良好，>5%现象级 |

## 💡 标题公式速查

### 模型1：利益结果型
```
零基础做到X
4个月做20万
踩坑XXX万，我复盘了X个关键点
选品秘籍、变现路径、清单
```

### 模型2：场景痛点型
```
普通人也能做到的X
不会XXX？这是我复盘的第一步
你为什么做了X篇都不涨粉？
选品不会做，是因为你忽略了这一步
```

### 模型3：新机会趋势型
```
史诗级更新来了
X行业又出现新玩法
这个AI+XXX居然能印钞？
没人注意到的X机会，其实很暴利
```

```
身份 + 我看到你的痛点 + 我能给你一个确定结果

```

## 🔧 AI写作流程

```
1. 找选题 → 选题库筛选
2. 定标题 → 套用标题模型
3. AI创作 → 调教角色+投喂框架
4. 编辑 → 去掉AI味+加个人案例
5. 排版 → 格式规范化
6. 发布 → 运营互动
```
