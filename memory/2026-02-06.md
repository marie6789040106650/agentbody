### 问题背景

**Server vs macOS 版本差异:**
```
Server:
├── Rust in platform-tools: 1.84.1 ✅
├── pinocchio: 0.10.2 (Cargo.lock) ✅
└── 编译: SUCCESS ✅

macOS:
├── Rust in platform-tools: 1.83 (已升级) ✅
├── pinocchio: 0.10.2 (crates.io 最新) ❌
└── 编译: FAILED (find_program_address 移除)

根因:
crates.io pinocchio 0.10.2 API 变更
solana-address 需要 curve25519 feature
```

### 解决方案

**Cargo.toml:**
```toml
solana-address = { version = "2.1", features = ["curve25519"] }
```

**代码更新:**
```rust
// 旧 API
pinocchio::Address::find_program_address()

// 新 API
use solana_address::Address;
Address::find_program_address(seeds, program_id)
```

### 执行命令

```bash
export SBF_SDK_PATH=/Users/bypasser/.cache/solana/v1.41/platform-tools/rust
cargo build-sbf
```
