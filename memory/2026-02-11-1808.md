assistant: ## Cron 是什么？

**Cron** 是一个 Linux/Unix 系统的**定时任务调度器**。

---

## 核心概念

| 概念 | 说明 |
|------|------|
| **Cron** | 定时任务服务（后台运行） |
| **Crontab** | Cron 的配置文件（cron table） |
| **Cron Job** | 定义好的定时任务 |

---

## 在 OpenClaw 中的作用

OpenClaw 使用 Cron 来**自动执行周期性任务**：

| 任务 | 频率 | 作用 |
|------|------|------|
| **检查更新** | 每5分钟 | 检查 OpenClaw 是否有新版本 |
| **检查 Notion** | 每周一9点 | 检查 Solana BootCamp 课程更新 |
| **优化检查** | 每月1号 | 执行优化检查脚本 |
| **Feedback 检查** | 每天 | 检查 openclaw-gen 项目的 GitHub 反馈 |

---

## 简单比喻

```
Cron = 你的"闹钟"
- 每天早上7点起床 → Cron 定时任务
- 每周一检查 Notion → OpenClaw Cron Job
```

---

## 在哪查看

**配置文件位置**:
```
~/.openclaw/cron/jobs.json
```

**查看当前任务**:
```bash
openclaw cron list
```

---

## OpenClaw 中的 4 个 Cron 任务

| 名称 | 频率 | 用途 |
|------|------|------|
| 检查更新 | 每5分钟 | 检查版本更新 |
| 检查 Notion | 每周一9点 | 课程更新通知 |
| 优化检查 | 每月1号 | 系统优化提醒 |
| Feedback 检查 | 每天 | GitHub 项目反馈 |

---

**简单说**：Cron 就是 OpenClaw 的"自动提醒助手"，帮你在指定时间自动执行任务！
